# Claude Hub 🔍

**Claude AI 확장 프로그램을 한눈에 찾아보세요!**

## 🚀 시작하기

### 웹에서 바로 사용하기
1. [Claude Hub](https://www.claude-hub.org) 방문
2. 검색창에 원하는 도구 입력 (예: "vscode", "agent", "monitoring")
3. 카테고리 필터로 결과 좁히기
4. 관심 있는 프로젝트 클릭하여 상세 정보 확인
5. "View on GitHub" 버튼으로 프로젝트 페이지 이동

### 로컬에서 실행하기
```bash
# 저장소 클론
git clone https://github.com/pinion05/claude-hub.git
cd claude-hub

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

[http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

## 💡 활용 팁

### 검색 팁
- **태그 검색**: #vscode, #terminal 등 태그로 검색

- `↑` `↓`: 검색 제안 탐색
- `Enter`: 선택 또는 검색
- `Esc`: 검색 제안 닫기
- `/`: 검색창 포커스 (계획 중)

## 🤝 기여하기

새로운 Claude 관련 프로젝트를 추가하고 싶으신가요?

### 빠른 추가 방법
1. [GitHub 저장소](https://github.com/pinion05/claude-hub) 방문
2. `storage/claude-hub-repositories/all-repositories.json` 편집
3. 아래 형식으로 프로젝트 추가:
```json
{
  "name": "owner/repo-name",
  "github_url": "https://github.com/owner/repo-name",
  "category": "카테고리명",
  "tags": ["태그1", "태그2"]
}
```
4. Pull Request 제출

### 프로젝트 추가 기준
- ✅ Claude AI와 직접적인 연관성
- ✅ 활발한 유지보수 (6개월 이내 업데이트)
- ✅ 명확한 문서화
- ✅ 오픈소스 라이선스
- ✅ 최소 10개 이상의 스타
